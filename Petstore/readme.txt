To run the code:

1. Open terminal and navigate to the directory Petstore if not already in
2. Open server.js and change port to any port between 5000-5500
3. Type in terminal "node server.js"
4. Install all dependencies with npm install
5. If one of the modules are not installing type npm install <here_is_the_module_Name>
4. Open and test the website on your browser
